    Overview


    powered by Pi-hole®

    FTLDNS™ (pihole-FTL) offers DNS services within the Pi-hole® project. It provides blazing fast DNS and DHCP services. It can also provide TFTP and more as the resolver part based on the popular dnsmasq. Furthermore, FTL offers an interactive API where extensive network analysis data and statistics may be queried.
